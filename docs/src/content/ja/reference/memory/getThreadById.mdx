# getThreadById リファレンス

`getThreadById` 関数は、ストレージからIDによって特定のスレッドを取得します。

## 使用例

```typescript
import { Memory } from "@mastra/core/memory";

const memory = new Memory(config);

const thread = await memory.getThreadById({ threadId: "thread-123" });
```
