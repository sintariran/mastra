# createThread

メモリシステム内で新しい会話スレッドを作成します。各スレッドは個別の会話またはコンテキストを表し、複数のメッセージを含むことができます。

## 使用例

```typescript
import { Memory } from "@mastra/memory";

const memory = new Memory({ /* config */ });
const thread = await memory.createThread({
  resourceId: "user-123",
  title: "Support Conversation",
  metadata: {
    category: "support",
    priority: "high"
  }
});
```
